To use this mock test server:

1. copy the mobiliser.war under Tomcat webapps
2. Start tomcat
3. Make sure client site has the following jar files:
wink-jettison-provider-1.1.2-incubating.jar
jackson-xc-1.7.0.jar
jackson-core-asl-1.7.0.jar
jackson-jaxrs-1.7.0.jar
4. Run Junit test to connect Tomcat server
